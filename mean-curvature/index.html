Curvature in Theory

May 13th, 2014

Introduction

Many Computer Science projects suffers from lack of theoretical information behind ideas which they are implement. This project is in fact different because has been built on simple idea that we would like to share not only code but also ideas behind it. This part of our web page contains basic explanation of several topic related to curvature and its application to smooth polyhedral meshes. The plan of this page is as fallow, at first we will introduce some notation and conditions which given mesh should fulfill to makes possible to calculate its curvature and in next section we will discuss properties of mean curvature and its application to smooth meshes. Finally the last part deal with drawbacks of mean curvature flow.

Meshes

We will right now give a definition of an abstract mesh and its properties and after that we will assume we always work with mesh which has same properties like this abstract one. Because we are interested only in meshes with each we can work with computers we will skip some general properties of abstract meshes like infinitely big set of vertices.

  1. Mesh is build from finit set of vertices - V where we will denot one vertex by pindex
  2. All vertices are connected by unordered edges which whole set will be denoted by E and particular edge denoted by e = (pi, pj) = (pj, pi) where pi, pj are distinct. We assume that the number of edges which go trought one vertex is finit and becouse we assumed in point 1 that set V is also finit therefore set E is also finit.
  3. We will introduce a set F which contais closed patches build from distinct vertices (p1, p2, …, pn, p1) where pi, pi+1 form an edge. We will call these closed patches faces.

Some extra conditions:

  1. Trought one vertex we can define 3 or more edges when vertex belongs to internal part of mesh or 2 when it belongs to boundary of mesh. (counter-example - figure 1. (a and b))
  2. Each edge is incident to 2 faces when its an internal edge or only one when edges is a boundary one. (counter-example - figure 1. (b and c)) In this case mesh is not 2-manifold and we will not interested in fallowed parts in this degenerated case.
  3. Each face has at least three edges. By simplification we will work in fallowed parts with faces which are triangles or quads.
  4. Each point belongs only to the one close mesh. This means that we can not have e.g. two close meshes connected by one point. (counter-example - figure 1. (d))

Mean curvature

Intuitive definition of curvature is how much something departs from being flat or straight. We can also define positive curvature e.g. sphere or negative e.g. saddle surface and also zero curvature when we talk e.g. about plane. In case of curves we can define curvature e.g. by osculating circle but this definition of curvature can not be used for surfaces.

Figure 1. From the left to right. Positive, negative and zero curvature. Image comes from thephysicsmill.com.

Curvature of surfaces can be defined in several ways e.g. Gaussian curvature or mean curvature, in this place we will take a look only on the second one therefore it has been used so far in FlowER. But because we are planning to add some new algorithms which use also Gaussian curvature, its definition should also appear in this place.

In continuous space mean curvature is defined as mean of principal curvature. We can understand this idea by taking a look on figure 2 which presents a surface with two principal curvature planes defined by normal to given point. To calculate min and max curvature we have calculate curvature of curves defined by surface and principal curvature planes in place where the each plane cross the surface.
