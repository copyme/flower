Mean curvature flow

Mean curvature flow of the cube
This movie presents a mean curvature flow of the cube with a time parameter equal to 0.05. Movie was rendered in the Blander 2.6 and OpenShot, also files which describe a geometry of a cube for each time were generated in upcoming open source software.

One of the first movies from first working versions. Movie was made in the Blender thus first version had not any support for drawing results.

First view on the FlowER
This movie presents an application which allows to study a flow of the given surface. This application is created especially for students of the discrete differential geometry and will be very simple thus we want to allow students to implement different kind of flows in a very simple way.

Movie presents first version which has a support for OpenGL.
