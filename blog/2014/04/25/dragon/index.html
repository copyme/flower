Dragon, Bunny and New Flows

Apr 25th, 2014

Last few days we sent on implementation of two new flows. Both are an extension of well known mean curvature flows. In the first case we have added a boundary case for meshes which are not close. Second case is a variation of mean curvature flow which is obtained by intensification of components of curvature vector by given power. As you can see e.g. for cylinder we obtained rounded result than for standard mean curvature flow.

Second news is related to movies which present mean curvature flow of more complex meshes. So in media category you can find movies which presents flow of well known Stanford dragon and bunny.

At the end we would like to inform you that we have gotten a feedback from one person about our project and this person pointed out that we should create a section on web page about theoretical aspects of flows. We think this is a good idea and we will work on this in the next few day. Stay tuned!
